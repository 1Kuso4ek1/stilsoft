Вопрос 1

    Вычислите следующее выражение:

    a) (5 > 3 && 4 < 8)
        Ответ: true

    b) (4 > 6 && true)
        Ответ: false

    c) (3 >= 3 || false)
        Ответ: true

    d) (true || false) ? 4 : 5
        Ответ: 4

Вопрос 2

    Оцените следующее выражение:

    a) 7 / 4
        Ответ: 1

    b) 14 % 5
        Ответ: 4

Вопрос 3

    Почему никогда не следует делать следующее:

    a) int y{ foo(++x, x) };
        Ответ: оператор инкремента применяет к x побочный эффект, поэтому аргументы функции могут быть вычислены в любом порядке

    b) double x{ 0.1 + 0.1 + 0.1 }; return (x == 0.3);
        Ответ: не стоит сравнивать плавающие числа оператором == из-за ошибки в вычислениях плавающих чисел

    c) int x{ 3 / 0 };
        Ответ: деление на ноль вызовет арифметическое исключение
